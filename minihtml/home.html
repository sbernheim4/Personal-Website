Projects

This Website

I built this website with HTML, NodeJS/ExpressJS, and SCSS using GulpJS as my task runner. The goal in mind was a quick, clean and simple website. I followed the BEM naming convention for my HTML and SCSS and used ExpressJS to enable compression and cacheing of static resources. All of this plus some additional optimizations got me a score of 100 on Google's Page Speed Test.

Kinetic Global

I, along with another developer, built the website for Kinetic Global. We used the MEAN stack (Angular 1) to develop it. It's hosted on Heroku and uses Amazon's S3 bucket for resource management. The website uses Sendgrid's API, and StripeJS to collect online donations. An online forum also integrates into Kinetic's Slack channels. In addition to building the website I also manage its SEO and got the website to be at the top when searching for Kinetic Global, Kinetic Think Tank and other similar terms.

Angular App

After building the Kinetic Global website, I wanted to build a base stack to review what I had just learned and built and to easily create new websites, essentially my own yeoman generator. That led me to building this app, a MEAN stack application but without the Mongo. I spent a lot of time determining the project's structure, essential npm modules needed and how the build process should work. It uses GulpJS for minification SCSS compilation, JS ES6 transpiling, livereloading and more.

Edelman for Brandeis

Shortly after building the Angular App a friend of mine told me he was running for Student Union President. When he asked if I could build his website I knew that this would be an ideal time to test the Angular App I had built. After about 3 hours, I had copied, formatted and designed the entire website, reviewed and revised it with him and had pushed it live onto a free heroku domain. Later that week he won the election.

M5 Systems

My first freelance work, I was hired to rebuild M5 Systems's website. M5 Systems is a consulting group that helps companies transition to the cloud. I used my Angular Stack project to build the app and it runs in an Amazon EC2 Instace. It uses Amazon's Simple Email Service and Nodemailer to handle form submissions. Working on this gave me the chance to finally delve into using AWS myself and after learning how to use it I transitioned my own website from Heroku to AWS
